## 🚀 دیپلوی روی Railway

### ۱. دکمه بالا رو بزن یا:

```bash
# Clone کن
git clone https://github.com/username/telegram-downloader-bot.git
cd telegram-downloader-bot

# Railway CLI نصب کن
npm install -g @railway/cli

# لاگین کن
railway login

# دیپلوی کن
railway up
